# polars-fastjson (core)

Pure-Rust core for [polars-fastjson](https://github.com/guywaldman/polars-fastjson).

This crate is usable standalone in Rust. The Python plugin lives in the
`polars-fastjson-plugin` crate and is shipped to PyPI via maturin.

It should be usable, however it's not intended to be publicly consumable - reach out if this is of interest to you.
